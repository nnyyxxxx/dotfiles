### Usage:
Install via curl
  ```shell
  curl -fsSL https://github.com/nnyyxxxx/hyprland/raw/main/install.sh | sh
  ```

Install via git
  ```shell
  git clone https://github.com/nnyyxxxx/hyprland
  cd hyprland
  chmod +x install.sh
  ./install.sh
  ```
